hexo个人网站配置流程

  • 安装必备依赖nodejs/hexo
  • 初始化目录 hexo init
  • 下载主题包,并安装相关依赖包
  • 修改主题配置文件和hexo配置文件
  • hexo s 生成本地预览
  • hexo g 生成静态文件
  • hexo d 上传到github上(需要 标准的仓库名称+ssh密钥+github用户密码)
